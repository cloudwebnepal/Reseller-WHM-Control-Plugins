# Reseller Controls (WHMCS Module)

Reseller Controls is a powerful WHMCS module designed to give hosting resellers direct access to essential WHM features without ever leaving the WHMCS client area. It removes the hassle of logging into WHM for common tasks and simplifies account management for you and your clients.

## 🚀 Features

### 🔄 Transfer cPanel Accounts from Remote Servers
Easily migrate full cPanel accounts from any remote WHM server. Just enter the source server details and let the module handle the entire transfer process automatically.

### ⌨️ Enable Shell (SSH) Access for cPanel Users
Allow your users to request or activate SSH access directly within WHMCS. Resellers can approve or enable shell access instantly using native WHM API integration.

### 🗑️ Delete DNS Zones
Clean up unused or outdated DNS zones with one click. Perfect for managing domains that have been removed, migrated, or reconfigured.

## ⭐ Why Reseller Controls?

- **Smooth and secure WHM API communication**
- **Reduces the need to login to WHM manually**
- **Gives resellers more control inside WHMCS**
- **Speeds up account setup, migration, and maintenance**
- **Simple UI designed for both new and advanced users**
- **A complete productivity booster for WHMCS-based reseller hosting providers.**
